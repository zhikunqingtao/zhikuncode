### 安全认证流程验证

```
请求 → RemoteAccessSecurityFilter
  ├── IP = 127.0.0.1/::1 → ✅ 直接放行 (层级 1)
  ├── IP = 私有网段 →
  │   ├── Cookie 有效 → ✅ 放行
  │   ├── Bearer Token 匹配 → ✅ 签发Cookie + 放行
  │   ├── URL ?token= 匹配 → ✅ 签发Cookie + 重定向
  │   └── 以上都不满足 → ❌ 401
  └── IP = 公网 → ❌ 403 (非私有网段拒绝)
```
